# 陈 文的个人网站

一个现代化、美观的个人作品集网站，展示我的技能、项目和联系方式。

## 🚀 特性

- **现代化设计**: 使用 Tailwind CSS 构建的响应式界面
- **深色模式**: 支持明暗主题切换，保护眼睛
- **流畅动画**: 丰富的过渡效果和微交互
- **移动端优化**: 完美适配各种设备尺寸
- **SEO 友好**: 语义化 HTML 结构
- **无障碍设计**: 遵循 Web 可访问性标准

## 🛠️ 技术栈

- **HTML5**: 语义化标记
- **Tailwind CSS**: 实用优先的 CSS 框架
- **JavaScript (ES6+)**: 现代化交互功能
- **Font Awesome**: 图标库
- **响应式设计**: 移动优先的设计理念

## 📁 项目结构

```
personal-website/
├── index.html          # 主页面
├── script.js           # JavaScript 交互逻辑
├── package.json        # 项目配置
└── README.md          # 项目说明
```

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

## 📱 响应式设计

网站采用移动优先的设计理念，完美适配：
- 📱 手机 (320px+)
- 📱 平板 (768px+)
- 💻 桌面 (1024px+)

## 🎯 交互特性

- **打字机效果**: 动态展示多个职业身份
- **视差滚动**: 英雄区域的视差效果
- **悬停动画**: 卡片和按钮的悬停状态
- **滚动动画**: 元素进入视口时的淡入效果

## 🔧 自定义配置

### 修改个人信息
在 `index.html` 中找到以下部分进行修改：
- 个人姓名和标题
- 关于我部分的内容
- 技能和经验等级
- 项目展示
- 联系信息

### 自定义样式
在 `<style>` 标签中修改：
- 颜色主题
- 动画效果
- 字体样式
- 间距和布局

### 添加新项目
在项目部分复制现有的项目卡片结构：
```html
<div class="group relative overflow-hidden rounded-xl shadow-lg card-hover cursor-pointer">
    <img src="项目图片" alt="项目名称">
    <div class="project-overlay">
        <!-- 项目信息 -->
    </div>
</div>
```

## 🌐 部署

### 静态托管
这个网站是纯静态的，可以部署到任何静态托管服务：
- **Vercel**: 连接 GitHub 仓库自动部署
- **Netlify**: 拖拽文件夹即可部署
- **GitHub Pages**: 免费的静态网站托管
- **Firebase Hosting**: Google 的静态托管服务

### 部署步骤
1. 将代码推送到 GitHub 仓库
2. 在托管平台连接仓库
3. 配置构建设置（如果需要）
4. 部署完成，获得在线链接
